# dots
My own dotfiles.

## How to use?

### Step 1: Install Homebrew
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
Run this command and do what script says.

### Step 2: Install Git
```bash
brew install git
```

### Step 3: Install Dotfiles
```bash
git clone https://github.com/UmutKzl/dots ~/dots
cd ~/dots
chmod +x bootstrap.sh
./bootstrap.sh
```
